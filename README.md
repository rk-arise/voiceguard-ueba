# 🛡️ VoiceGuard UEBA  
AI-Powered Voice Banking Security Platform  

## 🚀 Overview  
VoiceGuard UEBA is an AI-powered fraud prevention platform designed for **Bank of Baroda Hackathon 2025**.  
It leverages **voice biometrics, behavioral analytics, and telecom intelligence** to detect fraud **in real time during calls**.  

## ⚙️ Features  
- 🎤 Real-time voice biometric authentication  
- 🧠 AI-driven behavioral analytics  
- 📡 Telecom network intelligence  
- ⚡ Sub-second fraud risk scoring  
- 🌏 Multilingual support (22 Indian languages)
